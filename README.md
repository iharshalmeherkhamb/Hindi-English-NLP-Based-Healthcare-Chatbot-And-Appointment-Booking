# Hindi-English-NLP-Based-Healthcare-Chatbot-And-Appointment-Booking
Healthcare Chatbot: An AI-powered, multilingual disease prediction and doctor recommendation system. Built using Python, Streamlit, and RAG (Retrieval-Augmented Generation). It translates Hindi/English symptoms, predicts diseases using Vector Embeddings (all-MiniLM-L6-v2), and routes patients to the correct specialist.
